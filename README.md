## Object Form

    {
        "name": *name of the game,movie etc.*,
        "magnetLink": *magnet link of the torrent file*,
        "size": *file size*,
        "type": *torrent type (game, movie)*
        "additionalNote": *version number or installation tip (optional)*
    }
